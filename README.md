# IPL Data Analysis – Data Visualization Project
Analyzed cricket match data from the Indian Premier League to understand team performance, match statistics, and player achievements across multiple seasons. The project transformed raw IPL datasets into valuable insights using data analysis and visualization techniques.

# Technologies Used
Python | Pandas | NumPy | Matplotlib | Excel 

# Problem
IPL generates large volumes of match data each season. Without analysis, it is difficult to identify team performance trends, player achievements, venue statistics, and scoring patterns.

# Approach Used
• Team comparison analysis to evaluate win counts
• Venue analysis to identify stadium-wise match distribution
• Boundaries analysis to study total 4s and 6s by teams
• Player performance analysis across seasons
• Average run rate trend to study scoring patterns
• Player distribution analysis by country

# Outcome
The project revealed valuable trends in team performance, player achievements, and scoring behavior across IPL seasons, offering deeper insights into the tournament while further developing skills in data visualization and sports analytics.
